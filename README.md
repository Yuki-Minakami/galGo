# GalGo

**这是一个基于[Electron](http://electron.atom.io)的galgame引擎项目.**

对于galgame，最重要的工作是剧本和原画，对于广大galgame爱好者来说，程序不应该成为自行尝试的障碍。
本项目正是为解决上述问题而生

## 运行

- 要运行本项目，你需要安装[Node.js](https://nodejs.org/en/download/),下载本项目到本地
- 在目录下运行 **npm install && npm start**
